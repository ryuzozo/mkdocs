    JE/BE両刀サーバ

    JE/BE両方から参加できるサーバを建てたい
    OS:CentOS7.9
    java:java17
    directory:/mnt/xvdb/minecraft/

    java install

    1
    yum -y install java-17-openjdk java-17-openjdk-devel

    purpurmc install

    purpur download

    purpurmcからjarファイルをDL

    参考：Minecraftのサーバーの種類まとめ

    purpurmc start

    1
    java -Xms1G -Xmx2G -jar /mnt/xvdb/minecraft/purpur-1.18.2-1626.jar nogui

    参考：Minecraftサーバーにはメモリをどれくらい割り当てるべきなのか

    起動shell ※権限とかは適当に変える 

    1
2
    #!/bin/sh
java -Xms1G -Xmx2G -jar /mnt/xvdb/minecraft/purpur-1.18.2-1626.jar nogui

    初回起動後(必ず失敗する)に[/mnt/xvdb/minecraft/eula.txt]を編集
    [eula=false]を[eula=true]に変更

    GeyserMC/Floodgate install

    参考：【プラグイン紹介】統合版とJava版でクロスプレイが出来るプラグイン【GeyserMC】

    以下2ファイルをpluginsディレクトリに保存する
    GeyserMCからjarファイル(Geyser-Spigot.jar)をDL
    Floodgateからjarファイル(floodgate-spigot.jar)をDL

    1
2
3
    cd /mnt/xvdb/minecraft/plugins
wget https://ci.opencollab.dev//job/GeyserMC/job/Geyser/job/master/lastSuccessfulBuild/artifact/bootstrap/spigot/target/Geyser-Spigot.jar
wget https://ci.opencollab.dev/job/GeyserMC/job/Floodgate/job/master/lastSuccessfulBuild/artifact/spigot/target/floodgate-spigot.jar

    1度serverを起動しpluginsフォルダ内GeyserMCの[config.yml]編集する
    [auth-type=online]を[auth-type=floodgate]に変更 

    1
2
3
